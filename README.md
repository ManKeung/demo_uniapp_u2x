# 基础模板

uni-app uView2X开发基础配置模板

## 安装依赖

```bash
pnpm install
```

## 命令介绍

### war

h5页面打包war包

```bash
pnpm war
```

> war包名在`package.json配置字段"warName"默认"uview"`
> **打war包前先build项目**

### prepare

项目初始化

```bash
pnpm install
```

> `pnpm install`后会自动执行
